首先介绍下这个项目，该项目是一个纯净版不掺和任何业务的基础框架项目。
## 技术选型

### 管理
* maven依赖和项目管理
* git版本控制

### 后端
* IOC容器 Spring
* WEB框架 SpringMVC
* ORM框架 mybatis
* 验证框架 hibernate validator
* restful框架 jersey
* 缓存 ehcache
* 数据源 druid
* 日志 slf4j+logback
